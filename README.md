# 마켓컬리 clone 프로젝트

<br>

### 1. 프로젝트 Description

| 마켓컬리 웹사이트 디자인 클론 (https://www.kurly.com/main)

| 공통으로 사용되는 것은 component 폴더에, 나머지는 page에 있는 디렉터리 구조

<br>
<br>
<br>

### 2. 프로젝트 정보
- 프로젝트 구동
  ```
  > npm install
  ```
  ```
  > npm start
  ```
- 개발 스택

  라이브러리: ReactJS

  스타일: SCSS

<br>
<br>
<br>
